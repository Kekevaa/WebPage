ATM bank application

I was part of a project group whose mission was to make Bank ATM Application. Project included making qt creator application that works as a ATM machine. Main thing to do was making the databse and coding the modules for different functions so the ATM machine could work properly. I made RFID cardreader and corresponsive modules and also helped with other modules. Below is a GitHub link for the project.
