Spedens game

I was part of a project group whose mission was to make spedens game. Project included making the physical body for the game and soldering all the components together. Main thing to do was coding the modules for different functions so the game could work properly. I made highscore and sound modules and also helped with other modules. Unfrotunately this project was so early that we did not use GitHub yet.

Languages Used

Language Purpose
C++ Main logic for the game, including high score and sound modules
Arduino Used for interfacing with hardware components
